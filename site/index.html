Ok so what's this?

Just my silly hobby/pet project

One day I thought "hmm, what if I try out some data graph display thing with python & host it online", but then (as usual with pet projects) I got carried away and bought a domain.

What does a bored nerd do with these things?

1. a cheap domain and a raspberry pi
2. mediocre python skills, even worse webdev skills
3. near-infinite amount of free time (outside work)

You're looking at it! It's not entirely bad, right? :D
The page's files are hosted on github for easy maintaining :) I just pull the updated repo on my server when I'm done testing on my PC. Handy!

About the different pages:

About: You're here!

The cat page: Just a fun page about a certain cat

Cool data: A graph generated with python (pymatlab) displayed with a PHP script that fetches the latest graph based on timestamp in filename.
