Ok so what's this?

Just my silly hobby/pet project.

Basically, one day I thought "hmm, what if I try out a data visualization display thing with python & html.." but then (as usual with pet projects) I got carried away and bought a domain.

What does a bored nerd do with these things?

- cheap domain and a raspberry pi
- mediocre python skills, even worse webdev skills
near-infinite amount of free time (outside work)

You're looking at it! It's not entirely bad, right? :D
As a bonus funny, the page's files are hosted on github for easy maintaining :) I just pull the updated repo on my server when I'm done testing on my PC. Handy!

About the different pages:

About: You're here!

The cat page: Just a fun page about a certain cat

Cool data: A graph generated with python (pymatlab) displayed with a simple PHP script that fetches the latest generated graph (based on timestamp in filename). Data is collected from a CSV automatically, and the graph generation & display can be automated pretty easily. Kinda like a bootleg embedded Power BI report :D
